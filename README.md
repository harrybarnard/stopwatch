# Stopwatch
A JavaScript stopwatch application.

To view the poster load `<path to repo>/public/index.html` in your browser.

## Commands
Install build dependencies:
```bash
npm install
```
Build from source:
```bash
npm run build
```
Watch for source changes:
```bash
npm run watch
```
